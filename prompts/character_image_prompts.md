# Character Image Prompts for OpenArt AI

## How to Use These Prompts

Each prompt below is ready to copy-paste into OpenArt AI. Generate each character as a **reference image** first — you'll use these as visual anchors when generating scene clips later.

**Recommended settings:**
- Use the best available image model (e.g., SDXL, DALL-E 3, or Flux)
- Aspect ratio: **1:1** for character reference sheets
- Generate **2-4 variations** of each character to pick the best one
- Save your favorites in a `characters/images/` folder
